------------------------------------------------
config的三个作用域
缺省等同于local
git config --local 只对某个仓库有效
git config --global 对当前用户所有仓库有效
git config --system 对系统所有登录的用户有效

显示config的配置，加 --list
git config --list --local
git config --list --global
git config --list --system
------------------------------------------------
建git仓库
两种场景：
1.把已有的项目代码纳入git管理
cd 项目代码所在文件夹
git init

2.新建的项目直接使用git管理
cd 某个新建文件夹
git init your_project
cd your_project
